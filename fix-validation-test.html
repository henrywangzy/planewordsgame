🎯 朗读流程控制逻辑

新的游戏流程：

  1. 玩家击中目标敌机 → 触发 startCompleteWordReading()
  2. 开始4阶段朗读：英文单词 → 中文释义 → 英文例句 → 中文例句
  3. 朗读期间 wordReadingInProgress = true，暂停目标敌机生成
  4. 朗读完成后 → completeWordReadingFlow() → 3秒后自动切换下一个单词
  5. 切换新单词时重置所有朗读状态标志
